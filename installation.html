Installation¶

Usual PIP installation¶

pip install nctime

PIP installation from GitHub¶

pip install -e git://github.com/Prodiguer/nctime.git@master#egg=nctime

Installation from GitHub¶

  1. Clone our GitHub project:
git init
git clone git@github.com:Prodiguer/nctime.git
  1. Run the setup.py:
cd nctime
python setup.py install
  1. The nctime command-line is ready.

Dependencies¶

Linux distribution with Python 2.6+ is required. nctime uses the following basic Python libraries. Ensure that your Python environment includes:

nctime uses the following basic Python libraries includes in Python 2.5+. Becareful your Python environment includes:

  • os
  • re
  • sys
  • logging
  • argparse
  • importlib
  • ESGConfigParser
  • datetime
  • textwrap
  • multiprocessing
  • uuid
  • sqlite3

Some required libraries are not included in most Python distributions. Please install them using the usual PIP command:

  • netCDF4
  • netcdftime
  • nco
  • numpy
  • networkx

Warning

To support some corrections, NCO operators must be installed.
